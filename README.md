## Moodle Noodle
This repo is  primarily bootstap HTML snippets safe for use with the moodle LMS.
### HTML Directions
- Snippets are organised into folders
- Within each folder there are variations of each HTML object
- In your moodle LMS find the location you want to place your snippet
- from the repo click on the snippet you want, select the HTML from the code view window
- In moodle toogle the <> control to allow HTML input
- paste your snip
- toggle to preview view in moodle, check it is the HTML object you want
- If not delete and repeat with a different snippet
- If yes save and keep working on your project.
